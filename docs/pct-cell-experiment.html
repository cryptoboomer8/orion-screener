Pct Cell Variants

Four design options for the cascading percent-change row. Same five example patterns rendered each way — pick the look that reads cleanest for you. All examples use identical data so the only variable is the visual treatment.

Option A — 3-tier buckets

Three discrete color intensities per sign: faint (<1%), medium (1–5%), strong (>5%). No muddy middles, cells read as "levels" rather than a gradient.

Option B — Flat tint, text carries emphasis

Every cell gets the same subtle green/red fill. The number itself grows bolder and larger with magnitude. Pattern comes from text weight, not from background shade.

Option C — Flat tint + magnitude bar

Each cell has one fixed background tint. A thin colored bar at the bottom fills proportional to magnitude. The pattern reads as bar lengths — smooth continuous signal without muddy fill colors.

Option D — Plain colored text, no fills

Back to basics: no backgrounds at all, just green/red numbers with a thin underline. Smallest visual footprint. You lose the quick pattern recognition but the row is calmer.
